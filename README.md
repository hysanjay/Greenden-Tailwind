# Greenden-Tailwind
🌿 Greenden - Flower Shop Website

Greenden is a responsive flower shop website built using HTML and Tailwind CSS. The project showcases a simple and elegant design for an online flower store where users can explore products, view the homepage overview, and contact the store.

🚀 Features
🌸 Beautiful and modern flower shop UI
📱 Fully responsive design using Tailwind CSS
🏠 Home page with website overview
🛍️ Product page displaying flower collections
📞 Contact page for customer inquiries
⚡ Fast and lightweight static website
🛠️ Technologies Used
HTML5
Tailwind CSS
Responsive Web Design
